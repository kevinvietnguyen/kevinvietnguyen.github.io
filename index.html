PrescriptionPal

An application made in Android Studio using Java/XML that allows users to keep track of prescription regardless of their occupation. Whether it be a patient, pharmacy, or doctor. This application stores its information into its own database through Google's Firestore.

Mobster Massacre

A game made in Unity using C# scripts where the player acts as lone mob boss where an enemy gang has invaded your group's hideout. Hold out as long as you can to ensure as many of your mates can live! Be careful, more enemies may come over time!

The Last Claw
Shotgun Shellfish

A game made in Unity using C# scripts where the player will try their best to survive a "bullet hell". Many obstacles and enemies will come for the player. Take advantage of power-ups to live as long as you can!

  • GitHub

Balloon Ride

A game made in Visual Studio using C++ where the player's goal is to land on all three landing pads while using the limited gas that your air balloon has. Make sure you take in the nice view on your way!

  • GitHub

Tic Tac Toe

A reproduction of the classic game of Tic Tac Toe made in Eclipse using Java with knowledge of Swing. Similar to the classic game, the goal of the players is to compete against each other to see who is able to get three in a row first in a 3x3 grid.

  • GitHub

Interactive Ray Tracing

An application made in Visual Code using C++ where the user is able to create spheres and lighting on the fly. This allows users to have the ability to possibly create art that could be nice and appealing to look at.
